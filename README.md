# 📊 TCS Financial Analysis Dashboard (FY22–FY25)

## Overview

This project presents a comprehensive financial analysis of **Tata Consultancy Services (TCS)** using **Microsoft Excel**. The analysis covers four financial years (FY22–FY25) and focuses on evaluating the company's financial performance through key financial ratios, trend analysis, PivotTables, PivotCharts, KPI cards, and an interactive dashboard.

The objective of this project is to transform financial statement data into meaningful insights that support business decision-making and demonstrate practical financial analysis and Excel dashboarding skills.

---

# 🎯 Project Objective

* Analyze the financial performance of Tata Consultancy Services (TCS) from FY22 to FY25.
* Evaluate the company's profitability, liquidity, and financial position using key financial ratios.
* Perform trend analysis across multiple financial years.
* Build an interactive dashboard using Microsoft Excel.
* Generate business insights based on financial data.

---

# 📂 Data Source

The financial data used in this project has been collected from the audited **Tata Consultancy Services (TCS) Annual Reports** for the following financial years:

* FY2021–22
* FY2022–23
* FY2023–24
* FY2024–25

The analysis is based on information available in the Income Statement, Balance Sheet, and related financial disclosures.

---

# 📈 KPIs Analyzed

The dashboard includes the following Key Performance Indicators (KPIs):

* Revenue from Operations
* Profit After Tax (PAT)
* Total Assets
* Return on Equity (ROE)
* Return on Assets (ROA)
* Current Ratio

Additional analysis includes:

* Financial Ratio Analysis
* Common Size Analysis
* Trend Analysis
* Business Insights

# 🔍 Key Insights

* Revenue increased consistently from FY22 to FY25, indicating sustained business growth.
* Profit After Tax (PAT) showed steady growth, reflecting strong profitability.
* Total Assets increased year after year, demonstrating continuous business expansion.
* Return on Equity (ROE) improved throughout the analysis period, indicating efficient utilization of shareholders' equity.
* Return on Assets (ROA) increased steadily, highlighting effective asset management.
* The Current Ratio remained consistently above 3, indicating a strong liquidity position and effective working capital management.
